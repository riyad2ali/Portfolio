Automatic Rescue Breathing Unit

The Automatic Rescue Breathing Unit (ARBU) automatically delivers 100% oxygen rescue breaths to assist CPR respiration. The ARBU consist of an oxygen tank, pneumatic circuit board (PCB), Arduino motherboard, a case, and other accessories to assist oxygen travel to a patient. The software makes sure the valves deliver the correct amount of volume, pressure, and rate to the user. I embedded the Arduino software to the PCB software to work together and I worked on the code to ensure the patient is receiving the right amount of oxygen with the correct parameters.

Snake Game+

Created a snake game replica utilizing only Java to create the logic and reasoning to make the game function. The hardest part about creating this project was the issue with the UI lagging when the snake got bigger. I fixed this issue by deallocating the snake length array and removed accessing the same function repetitively by carrying snake length into memory and only accessing the function to make sure the snake moves together once every half-second.
